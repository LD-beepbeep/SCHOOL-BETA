# Contributing to Student OS 🎓

First off, thank you for considering contributing to **Student OS**! It’s community input that turns a good tool into a "perfect" dashboard.

## 🤝 How to Help
* **Bugs**: Open an issue with a screenshot and your browser info.
* **Features**: Want to add more features? Open an issue first so we can sync on the design.
* **PRs**: Submit your Pull Request against the `main` branch.

---
*By contributing, you agree to keep this project a respectful and helpful space for all students :)*
