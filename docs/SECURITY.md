# Security Policy

## Supported Versions
Currently, only the main branch of Student OS is supported with security updates.

## Reporting a Vulnerability
If you discover a security vulnerability (such as a way to bypass login or access other users' data), please **do not open a public issue.** Instead, please report it by:
1. Emailing [lars.dehairs@gmail.com]
2. Providing a description of the exploit and steps to reproduce it.

We will try to acknowledge your report within 48 hours and provide a fix as soon as possible!
